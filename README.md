# 🚀 Frontend UI Project with JavaScript, SCSS & Docker

## 📌 Overview
This project is a modern, interactive frontend web application built using **HTML, SCSS, and JavaScript**, and containerized using **Docker with NGINX** to simulate a real-world deployment environment.

It includes dynamic UI interactions, a task management system, and 3D hover effects to enhance user experience.

---

## ✨ Features

### 🎨 UI & Design
- Responsive layout
- SCSS-based styling (modular & maintainable)
- 3D hover effects on icons
- Smooth transitions and animations

### ⚡ JavaScript Functionality
- Add, delete, and complete tasks
- Search/filter tasks
- LocalStorage integration (data persists after refresh)
- Dynamic DOM manipulation

### 🐳 DevOps / Deployment
- Dockerized application
- Served using NGINX
- Easy local setup with Docker

---

## 🛠 Tech Stack

- HTML5  
- SCSS (SASS)  
- JavaScript (Vanilla JS)  
- Docker  
- NGINX  

---

## 🚀 How to Run the Project

### 🐳 Using Docker

1. Build the Docker image:

docker build -t frontend-app .


2. Run the container:

docker run -p 8080:80 frontend-app


3. Open in browser:

http://localhost:8080


---

### ⚡ Using Docker Compose


docker compose up


---

### 💻 Without Docker

Simply open:

index.html
